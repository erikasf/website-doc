HTML Structure

The template is based on Bootstrap Framework - http://twitter.github.com/bootstrap/. The default Bootstrap grid system utilizes 12 columns, making for a 1170px wide depending on your viewport. Below 767px viewports, the columns become fluid and stack vertically. Create a .row and add the appropriate number of .col-lg/.col-md/.col-sm/.col-xs Classes

   <div class="row">

      </div>

   </div>

</div>

Custom Css Changes : 

If you Want to change some design elements like color or want to hide some element without touching the less file just follow these steps.

For Example Consider in home page you want to temporarily hide the Shopping cart and my account section which is situated in the Top Bar.

       1. Open the CSS folder and look for the style sheet of with the name of your current color theme used for the template.
 
       2. Open the .CSS file in an editor of your choice.
 
       3. As Css file is commented, you can search with the element name or by using chrome development tool or firefox firebug you can find the exact line on which the elements CSS is that you want to change.
 
Using Chrome / firefox dev tool : Right click on the element and click inspect element, Note down the line number of element.

 

 

 

4. In your Editor go to the specific line number and add the desirable css property, in this case it will be 

 

display : none;

 

 

5. Save the file and view the changes.

Less Files and its Use

Less is a CSS Pre-Processor which helps in writing CSS with the use of dynamic functionalities like variable and mixins making CSS dynamic language where attributes can be called as variables and values can be passed through functions called mixins. for detailed documentation visit: http://lesscss.org/usage/

Testimonials Carousel

The Testimonial Carousel on the home page gives functionality of adding multiple testimonials and the images of the clients.

 

 

HTML Structure : 

 

<ul id="home-testimonials">

  <!--Testimonial Item-->    

   <li>

       <a class="testi-one testi-1" data-toggle="popover" data-placement="top" data-original-title=' Testimonial content  ' data-content='Name of client'>

       <img src="clianet image path" alt="" class="img-responsive client-logo-img" />

       </a>

    </li>

</ul>

 

For Adding Content of the testimonials you have to write in the data attribute data-original-title='' and the Names of client  in data attribute data-content=''

 

Actual Example : 

 

<li>

      <a class="testi-one testi-1" 
         data-toggle="popover" data-placement="top" data-original-title=' elementum id enim. Curabitur  Quisque velit nisi, pretium ut lacinia in, amet nisl tempus convallis quislacinia in, amet nisl tempus convallis quis ac lectus.' 
         data-content='<span class="testi-client-name">Jhon Snow</span> <br> <span class="testi-client-pos">Creative Officer</span>'>

  <img src="images/client-img-testi.png" alt="" class="img-responsive client-logo-img" />

    </a>

</li>

 

Jquery calling Function :

 

/* ==============================================
Home Page Testimonials
=============================================== */

    $("#home-testimonials").flexisel({
        visibleItems: 4,  // Items visible
        animationSpeed: 700,   // Moving animation speed
        autoPlay: false,  // Make Auto play true for starting it
        autoPlaySpeed: 3000,    // Auto play Speed        
        pauseOnHover: true,
        enableResponsiveBreakpoints: true,
        responsiveBreakpoints: { 
            portrait: { 
                changePoint:450,
                visibleItems: 1
            }, 
            landscape: { 
                changePoint:640,
                visibleItems: 2
            },
            tablet: { 
                changePoint:768,
                visibleItems: 3
            }
        }
    });

In The above Function you can manage the Speed of the carousel modifya attribute : 'animationSpeed', Number of client images visible modifya attribute : 'visibleItems' and autoplay modifya attribute : 'autoPlay'.

Setting Google Map Location

The Map location in the contact us Pages can be changed by modifying the Latitude and the Longitude co-ordinates which are present in the Jquery Function in the HTML files of Contact Us pages.

Steps to Change the Map Location : 

1. Get The Latitude and the Longitude of the location that you want to set in the map.

 

You can get the Latitude and Longitude of a location by searching here : http://itouchmap.com/latlong.html

 

2. Open the HTML page of contact US in Text editor.

3. At the bottom of the page There will be Jquery function for the map.

 

<script type="text/javascript">

    $("#map-canvas").gMap({
        styles:[{stylers:[
    {
        featureType: 'water', // set the water color
        elementType: 'geometry.fill', // apply the color only to the fill
        stylers: [
            { color: '#adc9b8' }
        ]
    },{
        featureType: 'landscape.natural', // set the natural landscape
        elementType: 'all',
        stylers: [
            { hue: '#809f80' },
            { lightness: -35 }
        ]
    }
    ,{
        featureType: 'poi', // set the point of interest
        elementType: 'geometry',
        stylers: [
            { hue: '#f9e0b7' },
            { lightness: 30 }
        ]
    },{
        featureType: 'road', // set the road
        elementType: 'geometry',
        stylers: [
            { hue: '#d5c18c' },
            { lightness: 14 }
        ]
    },{
        featureType: 'road.local', // set the local road
        elementType: 'all',
        stylers: [
            { hue: '#ffd7a6' },
            { saturation: 100 },
            { lightness: -12 }
        ]
    }
]}],
        controls: false,
        scrollwheel: false,
        maptype: 'ROADMAP',
        markers: [
            {
                latitude: 40.753317,
                longitude: -73.968905,
                icon: {
                    image: "assets/images/location.png",
                    iconsize: [85, 121],
                    iconanchor: [85, 121]
                }
            },
        ],
        icon: {
            image: "assets/images/location.png", 
            iconsize: [85, 121],
            iconanchor: [85, 121]
        },
        latitude: 40.753317,
        longitude: -73.968905,
        zoom: 12,
        mapTypeId: 'Styled'
    });

    </script>
